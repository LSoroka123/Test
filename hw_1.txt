2) Что такое HTTP и HTTPS 
HTTP - hyper text transfer protocol - набор правил передачи информациию Используется 80 порт.
HTTPS - набор правил для передачи информации засекреченый. Используется 443 порт по имолчанию
3)HTTP методы:
- GET - метод запроса информации от сервера
- POST - отправка полезной нагрузки на сервер
- PUT - загружает ресурс на сервер
- DELETE - удаляет ресурс
- OPTION - возвращает методы которые поддерживаются сервером
4)HTTP статус коды сервера:
100 - информационные
200 - успеха 200 - ок, 201 - created, 202 - accepted
300 - редиректы 301 - ресурс перемещен, 302 - found используется для перенаправления на другую страницу
400 - ошибки на кленте 400 - bad request (запрос составлен неверно), 401 - введен неправельный логин/пароль, 403 - ограничение в правах доступа, 404 - обращение к ресурсу которого нет на сервере
500 - ошибки на сервере 500 - server error(ошибка  не позволяет ответить на запрос), 503 - сервер недоступен по техническим причинам, 504 - gateway timeout - сервер не успел віполнить запрос за указаніц период
